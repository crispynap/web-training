# web-training
웹개발 연습용 저장소

## 사용언어
- 백엔드 : node.js + express(jade) + mysql
- 프론트엔드 : html + css + javascript

## 학습목표
- html, css, javascript를 주도적으로 사용해본다.
- flex 레이아웃의 사용을 몸에 익힌다.
- git과 github의 다양한 활용을 몸에 익힌다.

## 제작 계획
- 게시판 형태 홈페이지 제작

## 체크리스트
- 회원가입, 로그인, 로그아웃 구현
- 글 목록, 글 보기 구현
- 글 쓰기, 수정, 삭제 구현
- 헤더, 푸터 구성
- 프론트 페이지, 네비게이션 바 구성
- 에디터 미들웨어 활용
